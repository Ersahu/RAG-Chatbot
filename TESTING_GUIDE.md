## üêõ Troubleshooting

### **"Upload Failed"**

**Check:**
1. File format (PDF, DOCX, TXT only)
2. File size (< 10MB)
3. Backend is running
4. Check browser console for errors (F12)

**Fix:**
- Restart backend: `npm run dev` in backend folder
- Try smaller file
- Convert PDF to TXT if it's image-based

---

### **"No Answer Generated"**

**Possible Causes:**
1. Document not fully processed
2. Question not related to document content
3. Hugging Face rate limit exceeded

**Fix:**
- Wait for upload to complete
- Refresh page and try again
- Check backend logs for errors

---

### **"Rate Limit Exceeded"**

**Fix:**
- Wait 1 minute and try again
- Check Hugging Face usage: https://huggingface.co/settings/tokens
- Consider upgrading to Pro tier

## üîÑ How RAG Works (Behind the Scenes)

When you ask a question:

1. **Your Question** ‚Üí Converted to vector embedding
2. **Semantic Search** ‚Üí Finds 4 most relevant chunks from your documents
3. **Context Building** ‚Üí Combines relevant chunks
4. **LLM Prompt** ‚Üí Sends context + question to Mistral-7B
5. **Answer Generation** ‚Üí AI generates answer based only on your documents
6. **Source Attribution** ‚Üí Shows which documents were used
